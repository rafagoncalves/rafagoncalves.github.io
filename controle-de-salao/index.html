O web software Controle de Salão é um poderosa ferramenta de para administração de salões de beleza. O logotipo foi desenvolvido pensando em cores alegres e um visual moderno com um pattern poligonal compondo o símbolo, explorado, também, um espaço negativo contendo a silhueta de uma mulher. Foi desenvolvido, também, um trabalho em SEO, revisando os códigos do site, um folder para impressão como ferramenta auxiliar de venda e um email marketing.
